# InventoryWebProject

Spring Boot + Thymeleaf Inventory Management System (Add / Update / Delete / List)

How to run:
1. Install Java 17+ and Maven.
2. Open terminal in project root.
3. Run: mvn spring-boot:run
4. Open: http://localhost:8080
H2 console: http://localhost:8080/h2-console
JDBC URL (for H2 console): jdbc:h2:file:./data/inventory_db
